# hono-react-router-cf-template

A Cloudflare Workers template using Hono and React Router with TypeScript, Tailwind CSS, and D1 database integration.

We can deploy this app as a unified project. No need to split project and connect via service bindings.

## Development

Needs [mise](https://mise.jdx.dev/) installed.

```bash
mise install

mise run setup

mise run dev

mise run down
```
